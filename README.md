# 📋 Gerenciador de Tarefas

## 📌 Status do Projeto

🚧 Em desenvolvimento

## 💻 Tecnologias Utilizadas

* Java
* MySQL
* JDBC
* Swing
* Maven

## 👨‍💻 Time de Desenvolvedores

* Alexsandro Flores Rodrigues

## 🎯 Objetivo do Software

Sistema desktop para gerenciamento de tarefas e projetos, permitindo o controle de atividades, status e organização do fluxo de trabalho.

## ⚙️ Funcionalidades

* Cadastro de usuários
* Cadastro de projetos
* Cadastro de tarefas
* Listagem de usuários, projetos e tarefas
* Associação de tarefas a projetos
* Atualização de status de tarefas
* Interface gráfica com Java Swing

## 📂 Como Executar

1. Clonar o repositório
2. Abrir no NetBeans/Eclipse/IntelliJ
3. Executar a classe Main
